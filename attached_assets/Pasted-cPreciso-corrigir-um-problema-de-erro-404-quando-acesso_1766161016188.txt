cPreciso corrigir um problema de erro 404 quando acesso URLs que não são a página inicial do meu site.

PROBLEMA:
- URL principal funciona: https://rvmpersianas.app.br
- URLs internas retornam 404: https://rvmpersianas.app.br/jurere-internacional
- Erro ocorre ao acessar diretamente ou recarregar páginas internas

SOLUÇÃO NECESSÁRIA:

1. Configure o roteamento do projeto para funcionar corretamente com React Router (ou framework utilizado)

2. Adicione arquivo de configuração para redirecionar todas as rotas para index.html:
   - Se o projeto usa Vite, adicione configuração no vite.config.ts
   - Adicione arquivo vercel.json para deploy na Vercel
   - Adicione arquivo _redirects para Netlify
   - Adicione .htaccess para servidores Apache

3. Verifique se o BrowserRouter está configurado corretamente no App.tsx/main.tsx

4. Implemente a seguinte estrutura de arquivos de configuração:

vercel.json:
{
  "rewrites": [
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}

netlify.toml:
[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200

public/_redirects:
/*    /index.html   200

.htaccess (para Apache):
<IfModule mod_rewrite.c>
  RewriteEngine On
  RewriteBase /
  RewriteCond %{REQUEST_FILENAME} !-f
  RewriteCond %{REQUEST_FILENAME} !-d
  RewriteRule ^ index.html [QSA,L]
</IfModule>

5. Se estiver usando React Router v6, certifique-se de que:
   - BrowserRouter envolve toda a aplicação
   - As rotas estão definidas com <Routes> e <Route>
   - Não há HashRouter sendo usado (use BrowserRouter)

6. Adicione tratamento de rotas não encontradas:
   - Crie uma página 404 personalizada
   - Configure uma rota catch-all: <Route path="*" element={<NotFound />} />

REQUISITOS:
- Todas as rotas devem funcionar ao acessar diretamente pela URL
- O botão voltar/avançar do navegador deve funcionar normalmente
- Reload da página (F5) deve manter a rota atual
- Links internos devem funcionar sem recarregar a página
- Manter SEO-friendly URLs (sem # no caminho)
